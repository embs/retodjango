# reTODjangO

A TODO lists app based on React and Django.

## Featuring

- [x] Add tasks
- [x] Remove tasks
- [ ] Handle back-end errors
- [ ] List's editable title
- [ ] Editable tasks
- [x] Mark task as done
- [ ] Assign task's due date
- [ ] Multiple lists
- [ ] User accounts
  - Sign up & in
- [ ] Password recovery
- [ ] Lists ownership
- [ ] Tasks assignment
- [ ] E-mail daily report
- [ ] Download daily report as PDF
